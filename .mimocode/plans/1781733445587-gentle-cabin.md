## Task

## Production Page Architecture

The production page has **3 tabs** (Confection, Coupe, Sérigraphie), each rendering the same `ProductionTab` component with workshop-specific conditional sections.

### Tab Structure
- **Confection**: Chain info + KPIs + Gauges + Stoppages + OF Progress + SO Progress + Departage/Vignettes + Top Ops + WIP Flux + Efficience Opérateur + Trend + Methods KPIs
- **Coupe**: Chain info + KPIs + Gauges + Stoppages + OF Progress + SO Progress + Couverture Coupe + Chain Coverage + Tagging + OFs List + Qte Départage + Inline/Endline + Methods KPIs
- **Sérigraphie**: KPIs only + Couverture Sérigraphie + Flux Entrée/Sortie + Rejets + Methods KPIs
