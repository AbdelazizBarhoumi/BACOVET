# Logistics Page — Complete Reference Documentation

## Task
Create `LOGISTICS_SECTIONS_REFERENCE.md` — unified comprehensive reference for the logistics page with full server→view data flow for every section.

## Logistics Page Architecture

Single-page (no tabs), 8 sections (A through H):

| Row | Section | Endpoint |
|-----|---------|----------|
| A | Delivery Performance KPI cards (DOT, HOT, Respect Plan, Lead Time) | `GET /logistics/kpis` |
| B | Stock KPIs (Rotation, Stock Mort, Occupation) | `GET /logistics/stock-kpis` |
| C | Stock Composition Pie Charts (Provenance, Famille, Typologie) | `GET /logistics/stock-composition` |
| D | OF & Delivery (Livraison %, Délai Moyen) | `GET /logistics/ofs` |
| E | OF List table | `GET /logistics/ofs` (same) |
| F | Coverage bar charts (Chaîne, Coupe, Sérigraphie) | `GET /logistics/coverage` |
| G | Stock Reliability (Accessoires, Tissu, FG) | `GET /logistics/stock-reliability` |
| H | Stock search table (paginated, searchable) | `GET /logistics/stock-search` |
