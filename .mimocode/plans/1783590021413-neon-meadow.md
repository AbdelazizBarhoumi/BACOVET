## Root Cause Analysis

The page at `/v1/qualite` shows "ÔÇô" and "0 %" because the **database tables are empty** ÔÇö the backend controller logic is correct but has no data to work with.

### Why the page shows empty data

1. **Scheduler not running** ÔÇö `php artisan schedule:work` must be active for syncs to fire
2. **Novacity API** may be unreachable Ôćĺ `pieces_ok_jour`, `pieces_produites_jour`, `packets_rejetes`, `vw_defects` stay empty
3. **Google Drive mock server** (127.0.0.1:3002) may not be running Ôćĺ `sync_drive_br_*` tables stay empty
4. **`check_pass_qte` sync removed** (line 264 of SyncService: "Novacity endpoint returns 500 timeout") Ôćĺ QP Teams always empty

The controller logic is **correct** ÔÇö it gracefully returns null/grey when tables are empty.

   - No data seeded Ôćĺ all KPIs should have `status: 'grey'` and `value: null`

3. **`test_rft_jour_with_more_ok_than_produced_returns_null`**

4. **`test_br_bundling_zero_inspected_returns_grey`**
   - Seed `bundle_inspected = 0` Ôćĺ BR Bundling should be null/grey

5. **`test_br_print_zero_inspections_returns_grey`**

8. **`test_pareto_empty_when_no_defect_data`**
