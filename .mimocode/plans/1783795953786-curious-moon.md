# Fix Dropdown Open/Close Lag — Replace Radix Select with Lightweight Dropdown

## Root Cause

Clicking ANY dropdown still takes 3 seconds because of **Radix Select's Portal + Popper.js overhead**:

1. User clicks a `SelectTrigger`
2. Radix opens the dropdown → `SelectPrimitive.Content` renders inside `SelectPrimitive.Portal`
3. Portal creates a new DOM node at `document.body` (350+ portals already exist for all rows)
4. **Popper.js calculates floating positioning** — reads trigger dimensions, checks viewport collision, computes transform offsets
5. With 350+ portal nodes in the DOM, the browser's layout engine is overwhelmed by the positioning calculation
6. The 200ms CSS animation classes still fire (even with `animate-none`, the `tw-animate-css` custom properties are set)

The `FastSelectContent` fix removed animation classes but **did NOT remove the Portal or Popper.js** — those are the actual bottlenecks.

## Solution

Replace Radix `Select` with a lightweight custom dropdown that uses `position: fixed` instead of `SelectPrimitive.Portal`:

- **No Portal** — renders inline in the DOM
- **No Popper.js** — position calculated from `getBoundingClientRect()` once
- **No CSS animations** — instant open/close
- **No ScrollUpButton/ScrollDownButton** — simplified DOM

## Plan

### Step 1: Create `LightDropdown` component

**File:** `resources/js/routes-v1/pages/data.tsx`

A self-contained dropdown that renders its content with `position: fixed` based on the trigger's bounding rect. No Portal, no Popper.js, no animations.

```tsx
const LightDropdown = React.memo(function LightDropdown({
  value, onValueChange, disabled, className, placeholder, children,
}: { ... }) {
  const [open, setOpen] = useState(false);
  const triggerRef = useRef<HTMLButtonElement>(null);
  const contentRef = useRef<HTMLDivElement>(null);
  const [pos, setPos] = useState({ top: 0, left: 0, width: 0 });

  // Calculate position from trigger's bounding rect
  const updatePosition = useCallback(() => {
    if (!triggerRef.current) return;
    const rect = triggerRef.current.getBoundingClientRect();
    setPos({ top: rect.bottom + 2, left: rect.left, width: rect.width });
  }, []);

  // Open: calculate position, add listeners
  const handleOpen = useCallback(() => {
    updatePosition();
    setOpen(true);
  }, [updatePosition]);

  // Close on outside click or scroll
  useEffect(() => {
    if (!open) return;
    const handleClick = (e: MouseEvent) => {
      if (contentRef.current && !contentRef.current.contains(e.target as Node) &&
          triggerRef.current && !triggerRef.current.contains(e.target as Node)) {
        setOpen(false);
      }
    };
    const handleScroll = () => setOpen(false);
    document.addEventListener('mousedown', handleClick, true);
    document.addEventListener('scroll', handleScroll, true);
    return () => {
      document.removeEventListener('mousedown', handleClick, true);
      document.removeEventListener('scroll', handleScroll, true);
    };
  }, [open]);

  // Find label for current value
  const selectedLabel = useMemo(() => {
    // iterate children to find matching DataSelectItem
    // ...
  }, [value, children]);

  return (
    <>
      <button ref={triggerRef} onClick={handleOpen} disabled={disabled} className={...}>
        {selectedLabel || placeholder}
      </button>
      {open && (
        <div ref={contentRef} style={{ position: 'fixed', top: pos.top, left: pos.left, minWidth: pos.width, zIndex: 50 }}
             className="max-h-60 overflow-auto rounded-md border bg-popover shadow-md p-1">
          {React.Children.map(children, child => {
            if (React.isValidElement(child) && child.type === DataSelectItem) {
              const itemValue = (child.props as any).value;
              return React.cloneElement(child as React.ReactElement<any>, {
                onClick: () => { onValueChange(itemValue); setOpen(false); },
                selected: itemValue === value,
              });
            }
            return child; // pass through non-item children (search input etc.)
          })}
        </div>
      )}
    </>
  );
});
```

const DataSelect = React.memo(function DataSelect({ value, onValueChange, disabled, className, placeholder, children }) {

Remove the imports of `Select`, `SelectTrigger`, `SelectValue` from `@/components/ui/select` (no longer needed for this file).

### Step 3: Update `DataSelectItem` to work with `LightDropdown`

`DataSelectItem` needs to handle `onClick` and `selected` props from `LightDropdown`:

```tsx
const DataSelectItem = React.memo(function DataSelectItem({ value, className, children, selected, onClick }) {
  return (
    <div role="option" aria-selected={selected} onClick={onClick}
      className={cn("text-xs cursor-pointer rounded-sm px-2 py-1.5 hover:bg-accent ...", selected && "bg-accent", className)}>
      {children}
    </div>
  );
});
```

The `EndpointSelector` passes a `<div>` search input as a child of `DataSelect`. `LightDropdown` needs to render non-`DataSelectItem` children (like the search input) directly in the dropdown content, passing them through without modification.
