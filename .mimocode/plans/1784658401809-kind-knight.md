# Fix KPI "Afficher la cible" toggle

## Problem
The "Afficher la cible" toggle in the inspector doesn't show the target in the KPI widget.

## Root Cause
In `widgets/kpi.tsx` line 35: `{c.showTarget && tgt > 0 && ...}` — if `tgt` is 0 (or target is null/undefined), the target text is hidden even when the toggle is ON.

## Fix
In `resources/js/components/builder/widgets/kpi.tsx`:
- Change condition from `c.showTarget && tgt > 0` to `c.showTarget && c.target != null`
- This shows the target whenever it's explicitly set (even if 0)
- Keep the visual indicator (colored dot + text)

## Files Modified
- `resources/js/components/builder/widgets/kpi.tsx`

## Verification
1. Select F-REQ-102 as KPI → target should auto-fill to 5
2. Toggle "Afficher la cible" ON → "Objectif : 5%" should appear in the widget
3. Toggle OFF → target text should disappear
