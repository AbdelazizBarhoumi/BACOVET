# Plan: Per-widget outer spacing (top/bottom/left/right)

## Problem
The space between widgets comes from `react-grid-layout`'s global `margin={[8, 8]}` in canvas.tsx. Widget-level `marginTop/Bottom/Left/Right` from `boxStyle()` show 0px but don't affect inter-widget spacing because the grid's global margin adds space on top.

## Fix
1. Set `react-grid-layout` margin to `[0, 0]` — remove the global grid gap
2. Default each widget's `marginTop/Bottom/Left/Right` to 8 (from `STYLE_DEFAULTS` in store.tsx) so spacing is preserved
3. Inspector already has per-widget Haut/Bas/Gauche/Droite controls in the "Disposition" tab

## Files
- `canvas.tsx` line 279: change `margin={[8, 8]}` to `margin={[0, 0]}`
- `store.tsx` line ~55: change `STYLE_DEFAULTS` margins from 0 to 8

## Verification
Select a widget → adjust Haut/Bas/Gauche/Droite in inspector → only that widget's spacing changes
